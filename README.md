## frontend

    在和frontend同一级目录

    vite新建vue3工程
    npm create vue@latest
    cd frontend/
    npm install
    
    安装三方库
    npm install @vicons/ionicons5
    npm install @wangeditor/editor-for-vue@next --save
    npm install @wangeditor/editor --save
    npm i axios
    npm i naive-ui


## backend

    在backend目录下

    go mod init backend
    go mod tidy
